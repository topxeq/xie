# xie

#### Description
Xie语言（谢语言）是一门解释型编程语言（也叫脚本语言），最大的特色包括：跨平台；跨语言（目前仅支持Go语言）可嵌入；结合汇编语言和高级语言的优点；支持全中文编程（包括提示信息），语法简单易懂；单文件无依赖等。官网 xie.topget.org
